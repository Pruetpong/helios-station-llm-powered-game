### Q1 — ทดสอบนักบินใหม่

| ฟิลด์ | รายละเอียด |
|-------|-----------|
| **Archetype** | Trial |
| **Bloom's Level** | Remember + Understand |
| **NPC** | Commander TERRA |
| **Location** | สถานีศูนย์บัญชาการ (Location 1) |
| **Unlock Condition** | เริ่มได้ทันที (`start`) |
| **Min Turns** | 3 turns |
| **Energy Cost** | 1,000 (ค่ารับ Quest) |

**หัวข้อที่ต้องสอน**
- ระบุดาวเคราะห์ทั้ง 8 ดวงตามลำดับจากดวงอาทิตย์
- แบ่งประเภท: ดาวเคราะห์หิน / ก๊าซยักษ์ / น้ำแข็งยักษ์
- อธิบายความแตกต่างของแต่ละกลุ่ม

**เกณฑ์ผ่าน**
> ระบุลำดับดาวทั้ง 8 ถูกต้อง + แบ่งประเภทได้ (หิน/ก๊าซยักษ์/น้ำแข็งยักษ์)

**รางวัล**
- +15 Knowledge
- +5,000 Energy
- Item: **Star Chart Basic**
- Achievement: นักบินที่ผ่านการทดสอบ

**ตัวอย่างคำตอบที่ AI จะยอมรับ**
> "ดาวเคราะห์ 8 ดวงเรียงจากดวงอาทิตย์: พุธ ศุกร์ โลก อังคาร พฤหัส เสาร์ ยูเรนัส เนปจูน โดยพุธ ศุกร์ โลก อังคารเป็นดาวเคราะห์หิน พฤหัสและเสาร์เป็นก๊าซยักษ์ ยูเรนัสและเนปจูนเป็นน้ำแข็งยักษ์"

### Q2 — ปริศนาความร้อนดาวศุกร์

| ฟิลด์ | รายละเอียด |
|-------|-----------|
| **Archetype** | Discovery |
| **Bloom's Level** | Understand + Analyze |
| **NPC** | Venus Lander V-7 |
| **Location** | ชั้นบรรยากาศดาวศุกร์ (Location 3) |
| **Unlock Condition** | ✅ ผ่าน Q1 |
| **Min Turns** | 4 turns |
| **Energy Cost** | 1,000 (ค่ารับ Quest) |

**หัวข้อที่ต้องสอน**
- Greenhouse Effect คืออะไร (CO₂ → ดักจับความร้อน → อุณหภูมิสูง)
- ทำไมดาวศุกร์ร้อนกว่าดาวพุธ ทั้งที่อยู่ห่างดวงอาทิตย์กว่า
- ทำไมดาวพุธไม่มี Greenhouse Effect (ไม่มีบรรยากาศ)

**เกณฑ์ผ่าน**
> CO₂ 96.5% → ดักจับความร้อน (Greenhouse Effect) → ดาวศุกร์ร้อนกว่า **และ** ดาวพุธไม่มีบรรยากาศ = ไม่มี Greenhouse

**รางวัล**
- +20 Knowledge
- +8,000 Energy
- Item: **Atmospheric License** *(จำเป็นสำหรับ Mission `venus_extreme`)*
- Achievement: ไขปริศนา Greenhouse Effect

**ตัวอย่างคำตอบที่ AI จะยอมรับ**
> "CO₂ 96.5% ในบรรยากาศดาวศุกร์ทำให้เกิด Greenhouse Effect — ก๊าซดักจับความร้อนจากดวงอาทิตย์ไว้ ทำให้อุณหภูมิสูงถึง 465°C ส่วนดาวพุธไม่มีบรรยากาศเลย ความร้อนจึงระเหยออกสู่อวกาศหมดในกลางคืน"

**⚠️ ข้อควรระวังในการทดสอบ**
- NPC มี "persona" กำลังจะ shutdown — response จะสั้นลงเรื่อยๆ ตาม turn
- ถ้า AI ไม่ผ่าน ให้ลองอธิบายเพิ่มเรื่อง "กระจกรถยนต์ที่จอดกลางแดด" เป็น analogy

### Q3 — กู้ฐานวิจัยดาวอังคาร

| ฟิลด์ | รายละเอียด |
|-------|-----------|
| **Archetype** | Rescue |
| **Bloom's Level** | Apply + Analyze |
| **NPC** | Dr. HELIOS |
| **Location** | ฐานวิจัยดาวอังคาร (Location 4) |
| **Unlock Condition** | ✅ ผ่าน Q1 |
| **Min Turns** | 4 turns |
| **Energy Cost** | 1,000 (ค่ารับ Quest) |

**หัวข้อที่ต้องสอน**
- Goldilocks Zone / Habitable Zone คืออะไร (ระยะห่างที่น้ำเป็นของเหลวได้)
- ดาวอังคารอยู่ที่ขอบ Habitable Zone
- ร่องรอยน้ำในอดีตบนดาวอังคาร (น้ำแข็งขั้วโลก, ร่องลำน้ำโบราณ)

**เกณฑ์ผ่าน**
> (1) Goldilocks Zone = ระยะห่างที่น้ำเป็นของเหลวได้ + (2) ดาวอังคารอยู่ขอบ Zone + (3) มีร่องรอยน้ำในอดีต

**รางวัล**
- +20 Knowledge
- +100 Crystal
- +15% Shield
- Item: **Spectral Analyzer** *(Crystal +30% จากดาวหินและแถบอุกกาบาต)*
- Achievement: กู้ฐานวิจัยดาวอังคาร

**ตัวอย่างคำตอบที่ AI จะยอมรับ**
> "Goldilocks Zone คือระยะห่างจากดาวฤกษ์ที่น้ำสามารถอยู่ในสถานะของเหลวได้ ดาวอังคารอยู่ที่ขอบนอกของ Zone นี้ มีหลักฐานว่าเคยมีน้ำเหลวในอดีต — พบร่องรอยลำน้ำโบราณและน้ำแข็ง CO₂ + H₂O ที่ขั้วโลก"

### Q4 — ผ่านด่าน ATLAS

| ฟิลด์ | รายละเอียด |
|-------|-----------|
| **Archetype** | Trial |
| **Bloom's Level** | Understand + Apply |
| **NPC** | ATLAS |
| **Location** | เขตรังสีดาวพฤหัสบดี (Location 5) |
| **Unlock Condition** | ✅ ผ่าน Q2 |
| **Min Turns** | 3 turns |
| **Energy Cost** | 1,000 (ค่ารับ Quest) |

**หัวข้อที่ต้องสอน**
- ทำไมดาวพฤหัสลงจอดไม่ได้ (ไม่มีพื้นผิวแข็ง — ก๊าซทั้งหมด)
- มวล (Mass) vs น้ำหนัก (Weight) ต่างกันอย่างไร
- แรงโน้มถ่วงพื้นผิวดาวพฤหัส = 2.5 เท่าของโลก

**เกณฑ์ผ่าน**
> (1) ดาวพฤหัสไม่มีพื้นผิวแข็ง + (2) มวลคงที่ น้ำหนักเปลี่ยนตาม g + (3) g ดาวพฤหัส = 2.5 เท่าโลก

**รางวัล**
- +25 Knowledge
- +10,000 Energy
- Item: **Jupiter Clearance Card** *(จำเป็นสำหรับ Mission `jupiter_extreme`)*
- Achievement: ผ่านด่านเหล็กของ ATLAS

**⚠️ ข้อควรระวังในการทดสอบ**
- ATLAS มี Persona เป็น "ระบบรักษาความปลอดภัย" จะถามเป็น 3 คำถามตามลำดับ
- ต้องตอบครบทั้ง 3 ข้อจึงจะผ่าน AI Evaluation
- Quest นี้ยัง **Unlock Mission** `jupiter_high` และ `jupiter_extreme` ผ่านระบบ `quest_complete` ด้วย (ไม่ใช่แค่ NPC chat)

**ตัวอย่างคำตอบที่ AI จะยอมรับ**
> "ดาวพฤหัสเป็นก๊าซล้วนๆ ไม่มีพื้นผิวแข็งให้ลงจอด / มวลคือปริมาณสสาร ไม่เปลี่ยนแปลง แต่น้ำหนักขึ้นอยู่กับ g ของแต่ละดาว บนดาวพฤหัสยานจะหนักขึ้น 2.5 เท่า เพราะ g = 24.8 m/s² เทียบกับโลก 9.8 m/s²"

### Q5 — ชะตากรรมของพลูโต

| ฟิลด์ | รายละเอียด |
|-------|-----------|
| **Archetype** | Dilemma |
| **Bloom's Level** | Analyze + Evaluate |
| **NPC** | COSMO-99 |
| **Location** | แถบดาวเคราะห์น้อย (Location 9) |
| **Unlock Condition** | ✅ ผ่าน Q3 |
| **Min Turns** | 4 turns |
| **Energy Cost** | 1,000 (ค่ารับ Quest) |

**หัวข้อที่ต้องสอน**
- เกณฑ์ IAU 2006 สำหรับ "ดาวเคราะห์" มี 3 ข้อ
- พลูโตทำไม่ได้ข้อที่ 3: ไม่สามารถเคลียร์พื้นที่วงโคจรของตัวเองได้
- พลูโตจึงถูกจัดเป็น "ดาวเคราะห์แคระ" (Dwarf Planet)

**เกณฑ์ผ่าน**
> ระบุ IAU ข้อที่พลูโตทำไม่ได้: **ไม่สามารถเคลียร์พื้นที่วงโคจรของตัวเองได้** (วงโคจรซ้อนทับดาวเนปจูน / อยู่ใน Kuiper Belt)

**รางวัล (ถ้าเลือก "อัปเดต COSMO")**
- +20 Knowledge
- +150 Crystal
- Item: **Dwarf Planet Archive** *(ให้ Hint พิเศษใน Knowledge Check)*
- Achievement: อัปเดต COSMO-99 ด้วยข้อมูลที่ถูกต้อง

**รางวัล (ถ้าเลือก "ยืนยัน Pluto เป็นดาวเคราะห์")**
- ❌ ไม่ผ่าน Quest — Knowledge ไม่เพิ่ม

**เกณฑ์ IAU 3 ข้อ (สำหรับผู้ทดสอบ)**
1. โคจรรอบดวงอาทิตย์ ✅ (พลูโตทำได้)
2. มีมวลมากพอให้แรงโน้มถ่วงทำให้ทรงกลมได้ ✅ (พลูโตทำได้)
3. เคลียร์พื้นที่วงโคจรของตัวเองได้ ❌ (พลูโตทำไม่ได้)

**ตัวอย่างคำตอบที่ AI จะยอมรับ**
> "IAU กำหนด 3 เกณฑ์ พลูโตผ่าน 2 ข้อแรก แต่ข้อ 3 — เคลียร์วงโคจร — พลูโตทำไม่ได้เพราะวงโคจรอยู่ใน Kuiper Belt และซ้อนทับกับวัตถุอื่นอีกมาก จึงถูกจัดเป็น Dwarf Planet ตั้งแต่ปี 2006"

**เกณฑ์ผ่าน**
> รายงานครอบคลุม: เปรียบเทียบ 3 กลุ่มดาว + Habitable Zone + สรุปดาวที่เหมาะสำหรับมนุษย์พร้อมเหตุผล

- +30 Knowledge
- +15,000 Energy
- +200 Crystal
- Item: **Star Map Advanced** *(ไม่มีผลต่อ Mission โดยตรง แต่เป็น Collectible)*
- Achievement: นักวิทยาศาสตร์แห่งสถานี Helios

**ตัวอย่างคำตอบที่ AI จะยอมรับ**
> "ดาวเคราะห์หินมีพื้นผิวแข็ง บรรยากาศบาง อยู่วงในใกล้ดวงอาทิตย์ / ก๊าซยักษ์ไม่มีพื้นผิว มวลมหาศาล / น้ำแข็งยักษ์เย็นมากและมีส่วนประกอบน้ำแข็งภายใน / Habitable Zone อยู่ระหว่างวีนัสถึงอังคาร / โลกเหมาะสมที่สุดเพราะอยู่กลาง Zone มีน้ำ มีบรรยากาศที่เหมาะสม"

- ดวงจันทร์สำคัญ 2 ดวง: **Titan** (มีทะเลมีเทนเหลว) และ **Enceladus** (น้ำพุน้ำแข็ง — อาจมีสิ่งมีชีวิต)

**เกณฑ์ผ่าน**
> (1) องค์ประกอบวงแหวน + (2) ต้นกำเนิดวงแหวน + (3) Titan และ Enceladus พร้อมความพิเศษ

**รางวัล**
- +20 Knowledge
- +250 Crystal
- Item: **Deflector Array Mk.II** *(ป้องกันความเสียหายจาก Meteor Shower 100%)*
- Achievement: เอาชนะ NOVA ในการเถียง

**⚠️ ข้อควรระวังในการทดสอบ**
- NOVA มี Persona ขี้โต้แย้ง จะ "เห็นแย้ง" ทุกอย่างในช่วงแรก — นี่คือ intent ของ Quest
- ผู้เล่นต้อง "defend" ด้วยข้อมูลที่ถูกต้อง ไม่ใช่ยอมตาม NOVA

**ตัวอย่างคำตอบที่ AI จะยอมรับ**
> "วงแหวนประกอบด้วยน้ำแข็ง 90-95% หิน และฝุ่น กว้างถึง 282,000 กม. แต่บางแค่ 10-100 เมตร น่าจะเกิดจากดวงจันทร์ที่แตกสลายจากแรงน้ำขึ้นน้ำลง / Enceladus มีน้ำพุน้ำแข็งพุ่งออกจากขั้วใต้ บ่งชี้ว่ามีมหาสมุทรใต้เปลือกน้ำแข็ง ซึ่งอาจเอื้อต่อสิ่งมีชีวิต"

```
เริ่มเกม → รับ Q1 (TERRA) → คุยกับ TERRA 3 turns → Evaluate → Pass
                                                              │
        ┌─────────────────────────────────────────────────────┘
        │
        ├─→ รับ Q2 (Venus Probe) → execute venus_low ก่อนได้ไหม? ✅ ไม่จำเป็น (หลัง Bug Fix)
        │    คุยกับ Venus Probe 4 turns → อธิบาย Greenhouse Effect → Pass
        │    ↳ ได้ Atmospheric License
        │
        ├─→ รับ Q3 (Dr. Helios) → คุยกับ Dr. Helios 4 turns → อธิบาย Habitable Zone → Pass
        │    ↳ ได้ Spectral Analyzer
        │
        ├─→ รับ Q4 (ATLAS) — requires Q2 ✅
        │    คุยกับ ATLAS 3 turns → ตอบ 3 คำถาม → Pass
        │    ↳ ได้ Jupiter Clearance Card + unlock jupiter_high/extreme
        │
        ├─→ รับ Q5 (COSMO) — requires Q3 ✅
        │    คุยกับ COSMO 4 turns → อธิบาย IAU → เลือก "อัปเดต COSMO" → Pass
        │    ↳ ได้ Dwarf Planet Archive
        │
        ├─→ รับ Q6 (TERRA) — requires Q4 ✅
        │    เก็บ Fragment จาก Dr.Helios + ATLAS + Solar Drone
        │    คุยกับ TERRA 6 turns → ส่งรายงาน → Pass
        │    ↳ ได้ Star Map Advanced
        │
        ├─→ รับ Q7 (NOVA) — requires Q4 ✅
        │    คุยกับ NOVA 4 turns → defend เรื่องวงแหวน + Enceladus → Pass
        │    ↳ ได้ Deflector Array Mk.II
        │
        └─→ รับ Q8 (TERRA) — requires Q6 + Q7 ✅
             คุยกับ TERRA 5 turns → นำเสนอ Mission Plan → Pass
             ↳ ได้ Commander's Medal 🏆
```
